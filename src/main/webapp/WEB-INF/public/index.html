客户端类型：

高级 - 提供全套 Web 协同功能。该 Web 客户端最适合使用新版浏览器和更快的 Internet 连接。

标准 - 在 Internet 连接较慢、使用旧版浏览器，或为方便访问时，建议使用该类型。

移动 - 移动设备建议使用该类型。

要将首选的客户端类型设为默认，请在登录后前往“首选项”->“常规”选项卡更改选项中的符号。
请注意，所用浏流览器或显示器不能全面支持高级版本。 建议您使用标准版的客户端。
